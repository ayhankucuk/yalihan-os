# Memory — Oturum Hafızası

> Agent oturumları arasında bilgi paylaşımı için.

## Oturum Dosyaları

Oturum başına bir dosya:

```
memory/sessions/
  session-YYYY-MM-DD-{number}-{topic}.md
```

```markdown
# Oturum: 2026-06-25 - Session 33

## Öğrenilenler
-

## Devam Eden İşler
-

## Sonraki Adımlar
-
```
