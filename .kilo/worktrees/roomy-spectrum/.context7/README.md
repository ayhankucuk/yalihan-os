# Context7 Standards & Compliance

This directory contains Context7 canonical naming definitions, database schemas, and compliance standards for the Yalıhan Emlak platform.

## Canonical Field Name Conventions
- `status` → Use `yayin_durumu`
- `active` / `is_active` → Use `aktiflik_durumu`
- `email` → Use `eposta`
- `customer` / `customer_id` → Use `kisi` / `kisi_id`
- `order` / `sort_order` → Use `display_order`
- `latitude` / `longitude` → Use `lat` / `lng`
