## 1. KPI FRAMEWORK OVERVIEW

### 1.1 KPI Hierarchy

```
┌─────────────────────────────────────────────────────────────┐
│  KPI FRAMEWORK — 4-Tier Structure                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  TIER 1: NORTH STAR (1-2 Metrics)                         │
│  └── The one metric that predicts long-term success         │
│      → Net Promoter Score (NPS) — Customer loyalty         │
│      → Revenue Per Employee — Productivity                  │
│                                                             │
│  TIER 2: STRATEGIC (5-10 Metrics)                         │
│  └── Metrics that track progress toward strategic goals      │
│      → Revenue, Margin, Customer Acquisition Cost (CAC)    │
│      → Customer Lifetime Value (CLV), CLV:CAC Ratio        │
│                                                             │
│  TIER 3: TACTICAL (15-20 Metrics)                        │
│  └── Metrics that track operational health                   │
│      → Funnel metrics, Occupancy, ADR, Response time       │
│                                                             │
│  TIER 4: OPERATIONAL (50+ Metrics)                        │
│  └── Day-to-day metrics for teams                          │
│      → Individual KPIs per team, per process               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 North Star Metrics

```
┌─────────────────────────────────────────────────────────────┐
│  NORTH STAR METRICS — The Primary Success Indicators       │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  METRIC 1: NET PROMOTER SCORE (NPS)                      │
│  ├── Definition: "How likely are customers to recommend?"   │
│  ├── Target: >50 (Year 1), >60 (Year 2), >70 (Year 3)  │
│  ├── Measurement: Quarterly survey, real-time feedback     │

│  └── AI tracking: Sentiment analysis on all feedback      │

│  METRIC 2: REVENUE PER EMPLOYEE                          │
│  ├── Definition: Total revenue / FTE count                │
│  ├── Target: €100K (Y1), €150K (Y2), €200K (Y3)        │

│  └── Benchmark: Top real estate firms achieve €120K      │
```
